Sobre esta Aplicação

1. Arquitetura MVT

O Django utiliza a arquitetura Model-View-Template:

  • Model: Define a estrutura da base de dados (ex: Projetos, Tecnologias).
  • View: Processa a lógica e liga os dados aos templates.
  • Template: Interface final em HTML que o utilizador visualiza.

2. Vídeo Tutorial (CRUD de Projetos)

3. Tecnologias Utilizadas

Esta aplicação foi construída com:

  • Django (Backend & ORM)
  • SQLite (Base de Dados)
  • HTML/CSS (Frontend)
  • Git/GitHub (Controlo de Versão)
